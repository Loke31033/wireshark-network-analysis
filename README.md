# Wireshark Network Traffic Analysis

## Project Overview
This project demonstrates how to capture and analyze network traffic using Wireshark.

## Objectives
Capture network packets
Identify HTTP, DNS, ICMP, and SSH traffic
Detect reconnaissance activities such as port scanning

## Tools Used
Wireshark
Nmap
Linux Networking Tools

## Repository Structure

pcap_files
Captured network packets

screenshots
Packet analysis images

report
Investigation documentation

## Key Findings
HTTP and DNS traffic were successfully analyzed.
Port scanning behavior was detected using SYN packet patterns.
