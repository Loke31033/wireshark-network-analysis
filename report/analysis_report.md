# Wireshark Network Traffic Analysis

## Objective
Capture and analyze network traffic to understand common protocols and detect suspicious activity.

## Tools Used
Wireshark
Nmap
Ping Utility

## Protocol Analysis

### HTTP
Observed HTTP GET requests to web servers.

### DNS
DNS queries resolved domain names to IP addresses.

### ICMP
Ping traffic used to test network connectivity.

### SSH
Secure shell packets observed for remote access.

## Security Observation
SYN packets detected across multiple ports indicating possible port scanning activity.

## Conclusion
Wireshark enables deep inspection of network packets and helps identify potential security threats.
